[Events](../namespaces/Events.md).InteractionEvents

Represents interaction events. These events propagate to parents.

**`Typeparam`**

T - The primary target type.

**`Typeparam`**

R - The related target type.

**`Typeparam`**

RD - The related target type on drag events.
